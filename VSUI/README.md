VSUI
=====

### 概述

微商平台UI框架

### 使用

需要引用jQuery框架
```html
<link rel="stylesheet" href="./VSUI/vsui.min.css">
<script src="./VSUI/vsui.min.js"></script>

<script type="text/javascript">
    alert('alert');
</script>
```
